Create new:
rails new TDLv5_api -d postgresql
(do not use API only flag)

Create the Database
rails db:create

Gems:
bundle add bcrypt
bundle add rack-cors
bundle install
bundle update

react:
rails g model User email password_digest name

rails g model Entry content:text due_date:date user:references done:boolean
rails g controller Entry
rails g scaffold Tag name:string user:references 
rails g scaffold Tagging entry:references tag:references
